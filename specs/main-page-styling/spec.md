# Main Page Styling Specification

## User Stories

### User Story 1: Style Difference Section (P1)
**As a user,**
**I want the "difference between vibe coding and SDDRI" section on the main page to be visually appealing,**
**So that it clearly highlights the comparison and attracts attention.**

**Acceptance Criteria:**
- The div containing the "difference between vibe coding and SDDRI" section has a border.
- The border has a beautiful color.
- A hover effect is applied to this div.
- The "VS" div is visually centered within this bordered section.
